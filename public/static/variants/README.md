## Cómo Agregar Imágenes

1. **Coloca las imágenes** en esta carpeta con los nombres exactos:
   - **Grafito**: `Grafito1.jpg`, `Grafito2.jpg`, `Grafito3.jpg`, `Grafito4.jpg`
   - **Madera**: `Madera1.jpg`, `Madera2.jpg`, `Madera3.jpg`, `Madera4.jpg`
   - **Negro**: `Negro1.jpg`, `Negro2.jpg`, `Negro3.jpg`, `Negro4.jpg`
   - **Mármol**: `Marmol1.jpg`, `Marmol2.jpg`, `Marmol3.jpg`, `Marmol4.jpg`

2. **Recomendaciones para las imágenes:**
   - **Formato**: JPG o PNG
   - **Tamaño**: 150x150 píxeles (reducido para mejor rendimiento)
   - **Aspecto**: Cuadrado (1:1)
   - **Calidad**: Buena resolución
   - **Fondo**: Preferiblemente blanco o transparente

3. **Contenido sugerido para cada variante:**
   - **Imagen 1**: Vista frontal de la caja
   - **Imagen 2**: Vista lateral o ángulo
   - **Imagen 3**: Vista superior o detalle
   - **Imagen 4**: Textura o acabado específico

## Notas

- Si no se encuentra una imagen, se mostrará un placeholder gris
- Las imágenes se muestran en un grid de 2x2 por variante
- Tamaño reducido a 50x50px para mejor rendimiento
- El sistema es responsive y se adapta a diferentes tamaños de pantalla
- Efecto hover en cada imagen individual
